# algo-simu

to simulate steps of algorithms

# example

```
$ ./run.sh --checker ./examples/binarysearch-checker.sh 23 --simu _output/bin/algo-simu binarysearch 0 99
check 50 return no
check 25 return no
check 12 return yes
check 18 return yes
check 21 return yes
check 23 return yes
check 24 return no
23
```
